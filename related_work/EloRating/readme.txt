Descriptions of the m-files and databases

main.m : Run this file on MATLAB to perform Elo Rating method for patterned fabric defect detection

EloUpdate.m: Function to support main.m

PreProcessing.m: Function to support main.m

TestingStage.m: Function to support main.m

Database:
11 dot-patterned fabric images (5 defect-free, 6 defective)
